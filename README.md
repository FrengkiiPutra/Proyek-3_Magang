# Proyek-3_Magang
Repository ini berisi proyek Machine Learning untuk klasifikasi penyakit kardiovaskular menggunakan Python. Dengan dataset 70.000 data klinis, proyek ini membandingkan berbagai model seperti Logistic Regression dan Random Forest. Model terbaik mencapai akurasi 73,84% dan recall 68,44%, fokus pada peringatan dini medis yang presisi dan tervalidasi.
